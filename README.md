# Microservices Dashboard

## Running the Project

**Step 1:** Start all services
```bash
docker-compose up --build
```

**Step 2:** Open the dashboard
```
http://localhost:3000
```

**Step 3:** Register a new account or login

**Step 4:** Use the dashboard to manage orders and profile

## Features

**Authentication**
- User registration and login

**Profile Management**
- View and update profile

**Order Management**
- Create, edit, and cancel orders
- Update order status
- Search, filter, and paginate orders

## Tech Stack

Node.js + Express | Docker + Docker Compose | PostgreSQL | API Gateway
